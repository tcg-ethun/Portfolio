Html Introduction 🚀

আজ আমি আপনাদের জন্য একটি HTML টিউটোরিয়ালটি লিখছি , যা আপনাকে ৩০ দিনের কম সময়ে HTML শেখাতে সহায়তা করবে। আপনার শিক্ষাগত যোগ্যতার উপর ভিত্তি করে HTML শেখার একটি প্রস্তাবিত সময়সূচি নিচে দেওয়া হলো:

  • বিদ্যালয় পর্যায়ের শিক্ষার্থী : প্রায় ২৫ দিন |
  • উচ্চ বিদ্যালয় পাশ করা শিক্ষার্থী : প্রায় ২০ দিন |
  • কলেজ ও বিশ্ববিদ্যালয় পর্যায়ের শিক্ষার্থী : প্রায় ১০ থেকে ২০ দিন |


আপনি হয়তো ভাবছেন আমি কেন এই সময়সীমা নিয়ে কথা বলছি। এটা আমার জন্য গুরুত্বপূর্ণ, কারণ আমি চাই আপনি শেখার যাত্রা শুরু করার আগে আপনার প্রত্যাশাগুলো সঠিকভাবে নির্ধারণ করতে পারেন।


চলুন তাহলে HTML-এ ঝাঁপিয়ে পড়া যাক ! 😃



Html কি ? 🤔

এইচটিএমএল (HTML) বা হাইপারটেক্সট মার্কআপ ল্যাঙ্গুয়েজ ( Hypertext Markup Language ) হলো এমন একটি ভাষা যা ওয়েবসাইট তৈরির জন্য ব্যবহৃত হয়। এটি টিম বার্নার্স-লি ১৯৯১ সালে তৈরি করেন । এইচটিএমএল মূলত একটি মার্কআপ ভাষা, যার মাধ্যমে ওয়েব পেজে লেখার ধরন, শিরোনাম, ছবি, লিংক ইত্যাদি কীভাবে দেখানো হবে—তা নির্ধারণ করা হয়।


সংক্ষেপে বললে 🕦 :

  • এইচটিএমএল হলো ওয়েব ভাষা, যার মাধ্যমে ওয়েবসাইট তৈরি করা হয়।
  • এটি ওয়েব পেজের মৌলিক গঠন বা বিন্যাস নির্ধারণ করে।
  • এইচটিএমএল-এ একগুচ্ছ "ট্যাগ" থাকে, যা একটি HTML ডকুমেন্টে লিখতে সাহায্য করে এবং এই ধরনের ফাইল সাধারণত .html বা .htm File এক্সটেনশন দ্বারা সংরক্ষণ করা হয়।
  • এইচটিএমএলের একাধিক সংস্করণ আছে, যার মধ্যে HTML5 হলো সর্বশেষ ও আধুনিক সংস্করণ।


HTML এর বৈশিষ্ট্যসমূহ 🧬 :


  • এটি Platform-Independent : উদাহরণস্বরূপ, গুগল ক্রোম ব্রাউজারে উইন্ডোজে একটি ওয়েবপেজ যেমনভাবে দেখা যায়, ঠিক তেমনিভাবে ম্যাক বা লিনাক্সেও একদম একইভাবে দেখা যায়।

  • ওয়েবপেজে ছবি, ভিডিও ও অডিও যোগ করা যায়: যেমন ইউটিউব তাদের ওয়েবসাইটে ভিডিও দেখায়।

  • এইচটিএমএল একটি মার্কআপ ভাষা, প্রোগ্রামিং ভাষা নয় : অর্থাৎ , এটি নির্দেশনা দেয় কীভাবে ওয়েব কনটেন্ট উপস্থাপন করতে হবে, কিন্তু এটি লজিক বা গাণিতিক হিসাব করে না।

  • এটি অন্যান্য ভাষার সঙ্গে একত্রে কাজ করতে পারে : যেমন CSS, JavaScript ইত্যাদির সঙ্গে মিলিয়ে ইন্টারঅ্যাকটিভ বা গতিশীল (dynamic) ওয়েবপেজ তৈরি করা যায়।


কেনো এর নাম HyperText & Markup Language ? 🤔

  • "হাইপারটেক্সট" বলতে বোঝায় এমন লেখা বা টেক্সট, যা অন্য কোনো ডকুমেন্ট বা ওয়েবপেজের সঙ্গে সংযুক্ত থাকে (যেমন লিংক বা হাইপারলিংক)।

  • "মার্কআপ ভাষা" বলতে বোঝায় এমন একটি ভাষা, যেখানে নির্দিষ্ট কিছু ট্যাগ বা চিহ্ন ব্যবহার করে কনটেন্ট কেমনভাবে দেখানো হবে তা নির্ধারণ করা হয়।

    • তাই, এইচটিএমএল হলো একটি পদ্ধতি যার মাধ্যমে টেক্সট, ছবি, অডিও, ভিডিও ইত্যাদি নির্দিষ্টভাবে উপস্থাপন করা হয়, এবং এর জন্য বিশেষ ট্যাগ ব্যবহার করা হয়।


Note : 🗒️
ট্যাগ হলো অর্থবোধক শব্দ বা চিহ্ন, যেগুলো <...> কৌণিক বন্ধনীর মধ্যে লেখা হয়। যেমন: <P> (প্যারাগ্রাফের জন্য)। প্রতিটি ট্যাগের নিজস্ব অর্থ ও গুরুত্ব রয়েছে, যা ওয়েবপেজে বিভিন্নভাবে প্রভাব ফেলে।

HTML-এর ইতিহাস:

  • ১৯৮৯ সালে টিম বার্নার্স-লি (Tim Berners-Lee) বিশ্বব্যাপী ওয়েব (World Wide Web বা www) প্রতিষ্ঠা করেন, আর ১৯৯১ সালে তিনি HTML-এর প্রথম সংস্করণ তৈরি করেন।

  • ১৯৯৫ থেকে ১৯৯৭ সালের মধ্যে HTML-এর বিভিন্ন সংস্করণ উন্নয়ন ও পরিশোধনের কাজ চলতে থাকে।

  • ১৯৯৯ সালে একটি কমিটি গঠিত হয় যারা HTML 4.0-কে মান্যতা দেয় ও এটি এখনও অনেক জায়গায় ব্যবহৃত হয়। HTML-এর সর্বশেষ ও সবচেয়ে স্থিতিশীল সংস্করণ হলো HTML5।

HTML কীভাবে কাজ করে :

তুমি নিশ্চয়ই ফ্রন্টএন্ড ( Frontend ) আর ব্যাকএন্ড ( Backend ) শব্দদুটো শুনেছো।


ফ্রন্টএন্ড হচ্ছে ওয়েবসাইট বা অ্যাপের সেই অংশ, যেটা ব্যবহারকারীরা দেখতে পায় এবং যার সাথে তারা সরাসরি ইন্টারঅ্যাক্ট করে—যেমন: টেবিল, ছবি, বোতাম ইত্যাদি। এই অংশ তৈরি হয় HTML, CSS, এবং JavaScript দিয়ে।


অন্যদিকে, ব্যাকএন্ড হচ্ছে সেই অংশ যেটা পর্দার আড়ালে কাজ করে। ব্যবহারকারীরা ফ্রন্টএন্ডে যেভাবে ইন্টারঅ্যাক্ট করে, ব্যাকএন্ড সেই তথ্য সংরক্ষণ, বিশ্লেষণ ও প্রসেস করে। এটি সাধারণত Python, Ruby, Java ইত্যাদি প্রোগ্রামিং ভাষা ব্যবহার করে তৈরি হয়।


সংক্ষেপে বলা যায় :

  • ফ্রন্টএন্ড হচ্ছে যেটা ব্যবহারকারীরা দেখে ও ব্যবহার করে।

  • ব্যাকএন্ড হলো ওয়েবসাইট বা অ্যাপের সেই অংশ, যা পর্দার আড়ালে থেকে সমস্ত কার্যক্রম পরিচালনা করে।

ওয়েবসাইট কীভাবে কাজ করে ?

আমরা যখন ইন্টারনেটে কোনো তথ্য খুঁজতে চাই, তখন তা সার্চ করি ওয়েব ব্রাউজার ব্যবহার করে। ওয়েব ব্রাউজার সেই তথ্যটি ওয়েব সার্ভার থেকে নিয়ে আসে, যেখানে তথ্যগুলো HTML ডকুমেন্ট আকারে সংরক্ষিত থাকে।


একটি HTML ডকুমেন্ট তৈরি করা হয় কোড এডিটরে নির্দিষ্ট কিছু ট্যাগ ব্যবহার করে কোড লিখে। লেখার পর ফাইলটি “.html” এক্সটেনশন দিয়ে সংরক্ষণ করতে হয়।


সংরক্ষণ করার পর, ব্রাউজার সেই HTML ডকুমেন্টটি পড়তে পারে ও বিশ্লেষণ করতে পারে, এবং তারপর সেটিকে একটি সুন্দর ওয়েব পেজ আকারে আমাদের সামনে উপস্থাপন করে।


সারসংক্ষেপে :
  • আমরা ব্রাউজারে কোনো কিছু সার্চ করি।
  • ব্রাউজার ওয়েব সার্ভার থেকে HTML ফাইল নিয়ে আসে।
  • সেই HTML ফাইল ব্রাউজার পড়ে ও রেন্ডার করে, অর্থাৎ দেখায় আমাদের কাছে ওয়েবসাইট আকারে।

ওয়েব ব্রাউজার কী ?

ওয়েব ব্রাউজার হলো একটি প্রোগ্রাম, যা HTML ট্যাগ বুঝতে পারে এবং সেগুলোর ভিত্তিতে একটি ওয়েবপেজকে মানুষের পাঠযোগ্য ও দৃশ্যমান ফরম্যাটে প্রদর্শন করে।


ডেভেলপাররা HTML-এ কোড লেখে, কারণ এটি ব্রাউজারকে সহজভাবে নির্দেশ দেয় যে, কোন কনটেন্টটি কীভাবে দেখাতে হবে। যেমন: হেডিং, প্যারাগ্রাফ, ছবি, টেবিল ইত্যাদি।


ওয়েব ব্রাউজার এই HTML কোড পড়ে এবং সেই অনুযায়ী ওয়েবপেজটি তৈরি করে আমাদের দেখায়।

পরবর্তী অংশে আমি দেখাবো কীভাবে তুমি NotePad++ সেটআপ করে নিজের HTML কোড লিখতে পারো এবং তা ওয়েব ব্রাউজারে রান করিয়ে দেখতে পারো।


HTML ডকুমেন্ট কী?

একটি HTML ডকুমেন্ট হলো একটি টেক্সট ফাইল , যা “.html” বা “.htm” এক্সটেনশন দিয়ে সংরক্ষণ করা হয়। এই ডকুমেন্টের মধ্যে সাধারণ লেখা এবং কিছু নির্দিষ্ট ট্যাগ থাকে, যেগুলো < > এর মধ্যে লেখা হয়।


Rendered Page কী?

Rendered page হলো আমাদের HTML ডকুমেন্টের শেষ ফলাফল , অর্থাৎ ব্রাউজারে যে পেজটি আমরা দেখতে পাই সেটাই হলো রেন্ডার্ড পেজ।


একটা সাধারণ ওয়েবসাইট কীভাবে কাজ করে ?

  1. ওয়েব ব্রাউজার (যা ক্লায়েন্ট হিসেবে কাজ করে) ওয়েব সার্ভারের কাছে যেমন www.wikipedia.com এর মতো ওয়েবসাইটের অনুরোধ পাঠায়।


  2. ওয়েব সার্ভার সেই অনুরোধের জবাবে HTML, CSS, এবং JavaScript ফাইলগুলো ব্রাউজারের কাছে পাঠায়।


  3. এরপর ব্রাউজার এই HTML, CSS, এবং JavaScript ফাইলগুলো বিশ্লেষণ (parse) করে এবং তোমার সামনে একটি সুন্দর ওয়েবসাইট হিসেবে প্রদর্শন করে।


ওয়েব ব্রাউজার কীভাবে কাজ করে ?

ওয়েব ব্রাউজার ব্যবহারকারীর (client) কাছে HTML কোডকে পার্স করে এবং রেন্ডার করে দেখানোর গুরুত্বপূর্ণ কাজটি করে। এটি একটি জটিল প্রোগ্রাম, যা অনেক ধাপ পেছনে থেকে সম্পন্ন করে। নিচে সহজভাবে ব্যাখ্যা করা হলো :


ব্রাউজার কী করে ?

ব্রাউজার একটি অ্যাপ্লিকেশন, যা HTML ডকুমেন্ট পড়ে এবং সেটিকে ওয়েব পেজ হিসেবে প্রদর্শন করে। তবে, ব্রাউজার সরাসরি তথ্য সংগ্রহ করতে পারে না—এই কাজে সার্ভার মধ্যস্থতাকারী হিসেবে কাজ করে।


সার্ভার কী করে ?

যখন ব্রাউজার কোনো ওয়েবসাইটের অনুরোধ করে, সার্ভার সেই অনুরোধ গ্রহণ করে এবং HTML ডকুমেন্টটি ব্রাউজারের কাছে পাঠায়।


ব্রাউজারের দুটি প্রধান কাজ :

  1. Parsing (বিশ্লেষণ)

  2. Rendering (দেখানো)


Parsing কীভাবে হয় ?

  1. ব্রাউজার প্রথমে সার্ভার থেকে raw bytes (ডাটা) নেয়।

  2. এগুলোকে characters-এ রূপান্তর করে।

  3. এরপর character গুলোকে tokens-এ রূপান্তর করে।

  4. Tokens গুলো দিয়ে তৈরি হয় nodes .

  5. এই nodes গুলোকে গাছের মতো একটি গঠন বা DOM Tree ( Document Object Model ) আকারে সাজানো হয়।


Rendering কীভাবে হয় ?

একবার DOM Tree তৈরি হয়ে গেলে, ব্রাউজার প্রতিটি node-কে রেন্ডার করে স্ক্রিনে দেখায়। এর ফলে আমরা একটি পূর্ণাঙ্গ ওয়েবপেজ দেখতে পাই।


এখনই ব্রাউজার কীভাবে পুরোপুরি কাজ করে তা নিয়ে চিন্তা করার দরকার নেই। বরং, এখন HTML শেখার দিকেই মনোযোগ দাও। 😊

=> পরবর্তী টিউটোরিয়ালে আমরা Notepad++ ইনস্টল করবো এবং HTML কোড লিখবো । 😌

HTML Installation

এবার চলো আমরা কোডিংয়ের জন্য তৈরি হই। এই টিউটোরিয়ালে আমরা Notepad++ ইনস্টল করবো এবং HTML দিয়ে ডেভেলপমেন্ট শুরু করবো।


HTML শেখার জন্য পূর্বশর্ত ( Prerequisites ) কী ?

সোজা কথায়—কোনো পূর্বশর্ত নেই !

তুমি একেবারে নতুন হলেও HTML শিখতে পারো।


HTML হলো ওয়েবের ভাষা, এবং বেশিরভাগ ওয়েব ডেভেলপারের কোডিং শেখার প্রথম ধাপই হয় HTML থেকে। তাই ভয় না পেয়ে শুরু করে দাও — HTML শেখার জন্য আগের কোনো প্রোগ্রামিং অভিজ্ঞতা প্রয়োজন হয় না।


একটি HTML পেজ তৈরি করতে যেসব টুল প্রয়োজন :


1 . HTML এডিটর ( HTML Editor ) :

HTML লেখার জন্য একটি টেক্সট বা কোড এডিটর দরকার, যেখানে তুমি কোড লিখবে। এই টিউটোরিয়ালে আমরা ব্যবহার করবো Notepad++ কারণ এটি -

  • হালকা (lightweight)
  • ওপেন সোর্স

জনপ্রিয় HTML এডিটরসমূহ :
  • Text Editor : Notepad++, TextEdit (Mac)
  • Code Editor : Sublime Text, Visual Studio Code
  • IDE (Integrated Development Environment) : WebStorm, Eclipse
  • অনলাইন প্ল্যাটফর্ম : CodePen, JSFiddle

2. ওয়েব ব্রাউজার ( Browser ) :

ব্রাউজার HTML ট্যাগগুলো সরাসরি দেখায় না, বরং সেগুলো পড়বে (interpret করবে) এবং দৃশ্যমান ওয়েব পেজ হিসেবে উপস্থাপন করবে। এই টিউটোরিয়ালে আমরা ব্যবহার করবো Google Chrome।


    অন্যান্য জনপ্রিয় ব্রাউজার:

  1. Mozilla Firefox

  2. Opera Mini

  3. Safari ( For Mac )

  4. Microsoft Edge


সারসংক্ষেপে :

HTML শেখার জন্য তোমার দরকার হবে :

  1. একটি HTML এডিটর (যেমন VS Code)

  2. একটি ওয়েব ব্রাউজার (যেমন Chrome)


Installation & Setup of Notepad++ for HTML :

নিচের ভিডিও দেখে আপনি সহজে NotePad++ Download করতে পারবেন -




Notepad++ ডাউনলোড ও ইন্সটল করার ধাপসমূহ :

  1. ব্রাউজারে গিয়ে এই লিংকে প্রবেশ করো :
    🔗 Notepad++ Official Website

  2. সর্বশেষ ভার্সনটি খুঁজে বের করো এবং তাতে ক্লিক করো (যেমন: v8.6.5 বা নতুন যেটি রয়েছে)।

  3. পেজে গিয়ে নিচে স্ক্রল করে "Installer" অপশন খুঁজে বের করো (যেটায় .exe লেখা থাকে)।

  4. "Download 64-bit x64 Installer" বা তোমার সিস্টেম অনুযায়ী 32-bit installer-এ ক্লিক করো।

  5. ডাউনলোড শেষ হলে .exe ফাইলটি ডাবল ক্লিক করে চালু করো।

  6. ইন্সটলেশন উইজার্ড চালু হলে :

    • ভাষা নির্বাচন করো (English রাখাই ভালো)।
    • "Next" চাপো।
    • License Agreement এ "I Agree" চাপো।
    • ইন্সটল লোকেশন নির্বাচন করো (ডিফল্ট থাকলে ঠিক আছে)।
    • "Install" চাপো।
    • ইন্সটলেশন শেষ হলে "Run Notepad++" চেকবক্সটি টিক দিয়ে "Finish" চাপো।

এখন তুমি Notepad++ চালু করে HTML সহ যেকোনো কোড লেখা শুরু করতে পারো।
